# Asset setup (manual copy required)

Codex removed binary image files from this repository snapshot.

Before running or deploying, copy the landing page images from the original source folder:

```bash
cp -R "Mahjong Multiplayer Website Source/assets/." public/assets/
cp -R "Mahjong Multiplayer Website Source/uploads/." public/uploads/
```

These files are required by `public/landing-page.html`.
